fastascan

fastascan is the name of a tool that we had to create in the MSc in bioinformatics as an assignment for a Unix/algorithmics course. This was one midterm assignment after approximately 2 months using bash.

The purpose of this script is to design a tool that detects the .fa or .fasta files in an user-specified folder, does a summary of the files in the folder, and then an individual report containing the number of sequences, the number of amino acids/nucleotides in the whole file, specifying if the file is a symbolic link, and detecting if the file contains nucleotide or amino acid sequences.

The user can specify the folder to search in, and also if they want the first and last “n” lines of the file to be shown. If the number of lines of the file is smaller than 2n, the whole file is shown. The format to use the tool is fastascan [folder] [number_of_lines]. If the user doesn’t specify the folder to search, it will use by default the current directory. If the user doesn’t specify the number of lines, it won’t show any content of the file. If the 1st argument is an integer, and there is no 2nd argument, it will interpret the 1st as the number of lines that the user wants to see, and will search in the current directory.
